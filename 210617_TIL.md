#### SQL NULL 함수 관련
SQL에서 NULL관련 함수 사용 시 연산이 포함된다면 반드시 해당 함수로 시작할 필요 없음
```
SELECT 컬럼 + 컬럼 * NVL(컬럼명,NULL,0)
FROM DUAL;
```
